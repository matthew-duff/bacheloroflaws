### The Act contains targeted exceptions that are unusually specific compared with other literary works
None of these can be excluded by contract.

A. Backup Copy Exception
[[Copyright Act s 47C]] - Reproducing computer programs for backup
- **Purpose:** It is for personal use as a security spare or to replace a lost/damaged original.
- **Security:** It is part of a normal data backup routine.

**The Exception Fails if:**
- The original is an **illegal/infringing** copy
- The program is specifically **designed to prevent** copying.
- The user's **license has expired** or been terminated.

B. [[Copyright Act s 47B]] - Normal Use / Incidental Copying
It is **not** an infringement to make incidental copies (e.g., RAM loading) if:
- **Normal Use:** It happens automatically while running the program as designed.
- **Study:** It happens while running the program to understand how it works.
- **The Rule:** The original must be a legal copy and not used against express license terms.

C. [[Copyright Act s 47D]] Interoperability / Reverse Engineering
It is **not** a copyright infringement to reproduce or adapt a program to make it **compatible** with other software or hardware, provided:
- **The Goal:** To get information needed for a new product to "talk" to the original.
- **The Limit:** You only copy what is strictly necessary and only if that info isn't already available.
- **The Rule:** The original must not be a pirated copy.

D. [[Copyright Act s 47E]] - Error correction
It is **not** a copyright infringement to reproduce or adapt a program to **fix errors**, provided:
- **The Goal:** To make the program work as intended by the author or as described in its documentation (including hardware/software compatibility).
- **The Necessity:** You only modify/copy what is "reasonably necessary" to fix the bug.
- **The Market:** A working version of the program is **not available** to buy at a normal price within a reasonable timeframe.

E. [[Copyright Act s 47F]] - Security Testing
It is **not** a copyright infringement to reproduce or adapt a computer program for **good faith security testing**, provided:
- **The Goal:** It is strictly for testing security, investigating flaws, or fixing vulnerabilities.
- **The User:** It is performed by the owner or a lawful licensee
- **The Scope:** Only the "reasonably necessary" amount of copying is done.
- **The Constraint:** The required security information is not already available elsewhere.
